# TaskFlow API

Professional REST API for task management built with Flask.

## Features

- User authentication (JWT)
- CRUD for tasks
- Status filtering
- Clean architecture

## Tech Stack

- Python
- Flask
- SQLite (initially)
- JWT

## Installation

```bash
git clone https://github.com/MarioSatembo/taskflow-api.git
cd taskflow-api
pip install -r requirements.txt
python run.py
